- **Risk**: HIGH - Real money at stake
- **Use When**: After thorough testing in DEV/SIM modes

- **Behavior**: Simulates all transactions (dry-run)

- **Risk**: ZERO - No on-chain execution

#### ğŸ”µ SIM Mode
